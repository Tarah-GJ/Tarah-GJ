Using HTML selectors in CSS

CSS Selector Description Example
h1 {specific CSS code} Selects all <h1> elements. h1 {
color: blue; }
.class-name {specific CSS code} Selects all elements with the class "class-name". .class-name { background-color: #fff;}
#id-name {Specific CSS code} Selects the element with the ID "id-name". #id-name {
color: green; }
div p{Specific CSS code} Selects all <p> elements that are direct children of <div> elements. div p {
color: red; }
ul li{ } Selects all <li> elements that are descendants of <ul> elements. ul li {
color: blue; }
